# Cal. Eden: Next 🍃
Mental Arithmetic Training App — Cal. Eden: Next (Bloom Mode)

© 2026 ふる. All Rights Reserved.

---

## 🧠 概要

**Cal. Eden: Next** は、暗算力・処理速度・英語リスニングを同時に鍛えることを目的とした  
ブラウザ上で動作する複合学習アプリです。

- **Bloom Mode**：英語の文章問題に対して単位（個、リットル、キログラム、枚、メートルなど）が絡む計算問題を解く、英文理解と暗算を融合したモード

シンプルな構成ながら、集中力・ワーキングメモリ・暗算力を総合的に鍛える設計になっています。

---

## 🚀 使い方

1. `index.html` をブラウザで開くだけで動作します
2. インストール不要・サーバー不要
3. オフラインでも利用可能

---

## 📊 セッション情報表示

- 経過時間
